# Verkabelung

## Strip-Layout (Boustrophedon)

Die 10 Strip-Reihen werden im Schlangenverlauf verbunden:

```
Reihe 1: → → → → → → → → → →  ┐
                              │
Reihe 2: ← ← ← ← ← ← ← ← ← ← ←┘
        ┌──────────────────────
        │
Reihe 3: → → → → → → → → → →  ┐
        ...
```

Die U-Turn-Brücken liegen abwechselnd rechts und links im Lötkanal der Trägerplatte. Pro Reihenwechsel werden drei Drähte gelötet: 5V, GND, DOUT → DIN.
